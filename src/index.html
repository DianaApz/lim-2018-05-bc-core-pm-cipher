BIENVENIDOS A CIFRADO CESAR!

Si tienes algo confidencial que transmitir... usa el Cifrado Cesar ya que permite aumentar la seguridad de un mensaje, modificandolo para que ninguna palabra pueda entenderse y sólo quien conozca la clave pueda leerlo.

INSTRUCCIONES :


Para cifrar

Debes pensar en un número que funcionará como tu clave secreta y colocarlo.
Coloca el texto que desees cifrar ;es decir, la frase que escribas cambiara a otra frase.
Luego presiona el botón enviar y obtendras una frase nueva(frase cifrada).

Para descifrar

Debes colocar el número, que es tu clave secreta.
Coloca el texto que desees descifrar ;es decir, la frase cifrada cambiará a la frase original.
Luego presiona el botón enviar y obtendras la frase original.
Tip : Con tu clave secreta las personas que tu desees sabrán tu mensaje.
